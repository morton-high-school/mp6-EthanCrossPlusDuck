Requirements for an implementation of a singly linked list class LinkedList:

public LinkedList() - default constructor
public Node getFront() - returns the head of the linked list
public void append(Node node) - adds a node to the end of the linked list
public void insert(Node node, int index) - inserts a node at the given index or appends to the end if the index is out of bounds
public Node remove(int index) - removes and then returns the node at a given index, null if index out of bounds
public String toString() - returns information about the linked list
