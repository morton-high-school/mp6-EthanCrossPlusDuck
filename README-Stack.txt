Requirements for an implementation of the Stack class by use of the LinkedList class:

public Stack() - default constructor
public void push(Node item) - adds an item to the top of the stack
public Node pop() - removes and returns the item from the top of the stack
public Node peek() - returns the item from the top of the stack
public boolean empty() - returns true if the stack is empty and false otherwise
public int search(Node item) - returns the location of the item on the stack, -1 otherwise
