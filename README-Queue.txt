Requirements for an implementation of the Queue class by use of the LinkedList class:

public Queue() - default constructor
public void add(Node item) - adds an item to the back of the queue
public Node remove() - removes and returns the item from the front of the queue
public Node peek() - returns the item from the front of the queue
public boolean empty() - returns true if the queue is empty and false otherwise
public int search(Node item) - returns the location of the item in the queue
